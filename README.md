
# Stock Market Analysis & Prediction (ARIMA) 📈💰

(Time Series Analysis Project)

## 📌 Project Overview
A robust time-series forecasting model built to predict future stock prices using the **ARIMA (AutoRegressive Integrated Moving Average)** model. It identifies linear trends and seasonal patterns in historical market data to assist in informed decision-making.

## 🧠 Model Breakdown
*   **AR (Autoregression):** Uses the relationship between an observation and lagged observations.
*   **I (Integrated):** Employs differencing to make the time series stationary.
*   **MA (Moving Average):** Uses the dependency between an observation and a residual error from a moving average model applied to lagged observations.

## 🚀 Key Results
*   Successfully captured market volatility trends for major technology stocks.
*   Evaluated using RMSE and MAE metrics to ensure forecasting reliability.

## 🛠️ Tech Stack
*   **Language:** Python
*   **Statistical Modeling:** Statsmodels (ARIMA/SARIMA)
*   **Data Analysis:** Pandas, NumPy, yfinance
*   **Visualization:** Matplotlib, Seaborn

Open the `Stock_Analysis.ipynb` notebook to view the exploratory data analysis (EDA), stationarity tests (ADF test), and final price forecasts.
