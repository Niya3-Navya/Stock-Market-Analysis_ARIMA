
# Stock Market Analysis & Prediction (ARIMA) 📈💰

---Time Series Analysis Project

## 📌 Project Overview
A robust time-series forecasting model built to predict future stock prices using the **ARIMA (AutoRegressive Integrated Moving Average)** model. It identifies linear trends and seasonal patterns in historical market data to assist in informed decision-making.

## 🧠 Model Breakdown
*   **AR (Autoregression):** Uses the relationship between an observation and lagged observations.
*   **I (Integrated):** Employs differencing to make the time series stationary.
*   **MA (Moving Average):** Uses the dependency between an observation and a residual error from a moving average model applied to lagged observations.

## 🚀 Key Results
*   Successfully captured market volatility trends for major technology stocks.
*   Evaluated using RMSE and MAE metrics to ensure forecasting reliability.

## 🛠️ Tech Stack
*   **Language:** Python
*   **Statistical Modeling:** Statsmodels (ARIMA)
*   **Data Analysis:** Pandas, NumPy, yfinance
*   **Visualization:** Matplotlib
*   **Database:** Mongodb Compass

## ARIMA Statistical Modeling:-
 * Advanced Time-Series Modeling:
    This module utilizes the ARIMA (AutoRegressive
    Integrated Moving Average) statistical engine to perform a deep-dive into historical
    price lags and identify significant market trends.
 * Comparative Visual Analysis:
    The interface features a high-fidelity Plotly chart that
    overlays Actual Prices with Model Fitted Values, allowing users to visually validate the
    model's accuracy in capturing historical movements.
 * Statistical Diagnostic Metrics:
    A dedicated table presents critical model evaluation
    parameters, including AIC (Akaike Information Criterion), BIC (Bayesian Information
    Criterion), and Log Likelihood, to provide a quantitative measure of the model's
    performance.
 * Interpretative Support:
    To bridge the gap between complex statistics and user
    understanding, the dashboard includes informational callouts explaining how AIC and
    BIC values contribute to assessing the reliability of the mathematical fit.
 * Real-Time Computational Response:
    The statistical engine is fully dynamic, re-calculating all parameters and re-fitting
    the model instantly whenever the user modifies
    the stock ticker or the analysis time frame.
